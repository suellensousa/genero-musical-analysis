# Análise de Ouvintes por Gênero Musical

Este projeto realiza uma análise exploratória dos ouvintes de diferentes gêneros musicais com base em dados fictícios. O objetivo é entender a popularidade relativa de cada gênero e visualizar a distribuição dos ouvintes entre eles.

## Sobre

O projeto utiliza um conjunto de dados fictícios que inclui informações sobre diferentes gêneros musicais e o número de ouvintes (em milhões) para cada gênero. A análise foca em entender quais gêneros são mais populares e como os ouvintes estão distribuídos entre esses gêneros.

## Tecnologias Utilizadas

- **Python**: Linguagem de programação utilizada.
- **Pandas**: Biblioteca utilizada para manipulação e análise dos dados.
- **Matplotlib**: Biblioteca utilizada para criar visualizações gráficas.

- ## Análise dos Resultados

- **Gênero mais popular:** Sertanejo é o gênero com o maior número de ouvintes, seguido por Pop e Funk.
- **Gênero menos popular:** Eletrônica tem o menor número de ouvintes, sugerindo uma base de fãs menor.
- **Distribuição:** A análise visual mostra que os ouvintes estão concentrados em alguns gêneros principais, com uma queda gradual em gêneros menos populares.

### Visualizações Incluídas:

1. **Gráfico de Barras:** Mostra a quantidade de ouvintes por gênero musical.
2. **Gráfico de Pizza:** Exibe a proporção de ouvintes entre os diferentes gêneros.
